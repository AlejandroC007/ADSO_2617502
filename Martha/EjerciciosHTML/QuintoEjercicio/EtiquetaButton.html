Etiqueta Button

La función principal de la etiqueta "button" es permitir al usuario hacer clic en un botón cualquiera y activar una acción, como enviar un formulario o realizar una acción específica en la página web. Los botones también se pueden utilizar para mejorar la experiencia del usuario, como enlaces a otras páginas o para resaltar elementos importantes en una página web.

La etiqueta "button" se puede personalizar con diferentes atributos para cambiar el aspecto y el comportamiento del botón, como el texto del botón, la acción que se realiza al hacer clic, el estilo de diseño y la ubicación en la página web.
